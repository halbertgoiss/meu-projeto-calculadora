# Calculadora em Python

Uma calculadora simples com testes unitários.

## Como usar

Instale as dependências:

```bash
pip install pytest
```

Execute os testes:

```bash
pytest test_calculadora.py -v
```

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Potência
- Raiz Quadrada

## Arquivos

- `calculadora.py` - Funções da calculadora
- `test_calculadora.py` - Testes
- `README.md` - Documentação
